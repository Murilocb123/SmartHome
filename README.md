# Smart Home
## Descrição
Projeto desenvolvido para atividade na materia de *"Linguagem de Programação e Paradigmas"*

## Como usar
- Requisitos:
    - Node: v20.17.0

- Inicializando o projeto:
    - Clone o repositório e execute linha a linha:
    ```bash	
    > git clone {url}

    > cd smart-home
    > cd backend
    > npm install
    > cd ..
    > cd frontend
    > npm install
    > npm start
    ```
    - Acesse a pasta do backend e execute o comando:
    ```bash
    > npm start
    ```
